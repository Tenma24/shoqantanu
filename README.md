# Шоқантану Professional Site V4

Бұл нұсқада карта толық түзетілді.

## Негізгі түзету
- Карта енді сыртқы интернетке тәуелді емес.
- `assets/maps/shoqan-route-map.png` негізгі карта ретінде бірден көрінеді.
- `#routeMap` SVG қабаты оның үстінде қозғалыс пен нүктелерді көрсетеді.
- Leaflet CDN алып тасталды, сондықтан үлкен бос пробел шықпайды.

## Қалай ашу керек
1. ZIP файлын шығарыңыз.
2. `shoqantanu_pro_site_v4/index.html` файлын браузерде ашыңыз.
3. Карта бөлімінде батырмалар, slider және нүктелер жұмыс істейді.
